Report on LPG

Looking after the consumer dataset we observed that the recorded entries were only for one weekâ€™s timestamp. It was difficult to use such data set for future forecasting as the data was recorded per second so we used this data only for analysis purpose and moved ahead for future prediction using the monthly data.

The monthly data were recorded for each day. To get a fair idea about what the data tells us we decomposed it using Time Series Techniques, namely trend, seasonality and residual charts. We noticed that the data has a white noise present which implies high residuals and randomness. Therefore, no seasonality and trend.

As there was too much randomness in the data we opted for neural networks. In Neural Network we used Long short-term memory (LSTM), which is an artificial recurrent neural network (RNN) architecture used in the field of deep learning which is capable of learning long dependencies (previous data).Using LSTM we tried to create a time series model which was able to learn the pattern and accordingly predict the LPG usage consumption values.

The alternate model used was SARIMA, which is one of the most used techniques in time series forecasting. SARIMA is the model which is capable of capturing trend and seasonality. We used LSTM to evaluate the working of SARIMA. Further, we took a cumulative of all the days for each month and monthly usage data and we predicted the future demand for 5 months.
