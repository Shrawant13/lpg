LPG Analysis

Analysis performed on gas usage consumption for a week, months and year on consumers LPG usage overtime under the dealers.

The Dataset consists of two different files - dealers dataset and consumer dataset. Dealers Dataset - The dealer's dataset has recorded entries about the information from the dealer's end which covers the number of bookings supplied on each day from 2013 to 2017.Consumer Dataset -The consumer dataset has recorded entries for the usage in grams of LPG and the timestamp from the year 2018 to 2019. This data is only recorded for 3 consumers from the dealer1. Consumer data set also have 2 subfiles weekly usages recorded for each minute and monthly usages recorded for each day.
